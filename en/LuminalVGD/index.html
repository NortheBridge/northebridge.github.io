LuminalVGD

The Luminal Video Graphics Display Driver — a first-party, attestation-signed virtual display driver for LuminalShine, written in Rust for Windows 11 and Windows 11 Insider Preview.

Status: Pre-development

Features

Rust, from the ground up

LuminalVGD marks the Foundation’s transition to Rust for driver development — bringing memory safety to the code that sits closest to the display stack.

Legitimately signed, with attestation

A properly attestation-signed driver — like SudoVDA, and unlike pf-display — so it installs on stock Windows 11 with no test-signing mode or security compromises.

Modern IddCx architecture

A UMDF Indirect Display Driver (IddCx) built specifically for Windows 11 and Insider Preview builds — running in user mode, not the kernel.

Per-client virtual monitors

Exact mode lists per streaming session — resolution, refresh rate, and HDR10 metadata matched to each client, including frame-generation-aware refresh doubling.

First-class recovery

Designed to survive GPU TDRs and driver restarts without wedging WUDFHost, with a control interface built for LuminalShine’s watchdog and recovery ladder.

Built for LuminalShine

Render-adapter selection on hybrid-GPU laptops, signed and serviced through the LuminalShine installer, and set to supersede SudoVDA as the default virtual display backend.
